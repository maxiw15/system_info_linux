<h2>Часть 3. Считывание данных из ОС Linux
Linux является одной из популярных систем. Множество инцидентов
происходит с использование Linux машин. В ходе работы необходимо
отразить информацию из пунктов ниже.</h2>
1. Данные о хосте (дата, имя хоста, ip адресацию, ядро ОС, список
пользователей, uptime, список последних входов в систему, перечень
текущих сессий, привязка пользователей к группам).<br>
2. Информация о директориях пользователей и рута. Здесь также нужно
показать краткий вывод структур директорий.<br>
3. Папки, куда данные программы могут писать полезную инфу, например
хранить базы, пароли, историю и прочие сведения. <br>
4. Браузерные базы, где хранятся настройки, пароли, закладки и история
посещений. <br>
5. Наличие wine и перечень ПО установленного там <br>
6. История консоли <br>
7. Информация об обращении к сети TOR <br>
8. Перечень процессов системы, в том числе в состоянии демона <br>
9. В скрытых файлах (в Linux начинаются с точки) в домашних каталогах
пользователей хранятся файлы разных конфигов, они могут содержать
полезную информацию о последней активности, сохраненных сессиях и
истории команд, набранных в терминале <br>
